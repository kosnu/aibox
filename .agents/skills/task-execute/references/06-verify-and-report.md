# Verify And Report

Run relevant checks for the affected area after review fixes are applied.

Treat completion as "no convincing contradiction found," not merely "tests passed."

## Final Report Order

Summarize:

1. outcome and changed behavior shipped
2. verification run
3. review findings fixed or remaining risks
4. review budget used and why it fit the size/risk classification
5. final synchronization status of each representation
6. representations checked and intentionally unchanged
7. minor scope clarifications handled without re-approval, if any

Keep the final report concise. Do not reconstruct the original plan, list every inspected file, or propose a new plan unless execution stopped for re-approval.
